📖 LeadScope AI — Interactive User Guide

Understand the 3-Step automated workflow cycle that powers LeadScope and helps you close local business clients with ease.

01

🔍 Scan & Discover Leads

Enter a local business type (like bakery) and location (like Paris) in the Lead Finder screen.

  • Scrapes listings directly from Google Maps.
  • Automatically discards businesses that already have custom websites.
  • Enriches leads with contact channels (Facebook, Instagram, LinkedIn, WhatsApp).
  • Queries search engines for public business emails directly.
02

🌐 Pitch Dynamic Proposals

Review results and pitch customized website templates. Link templates directly from your GitHub templates folder.

  • Typeless generation: templates load from your GitHub repo on-the-fly.
  • Automatically injects business contact details into placeholders.
  • Embeds a sticky checkout banner linking to your Fiverr and WhatsApp.
  • Adaptive pitching hooks built directly from your CRM checklist status.
03

🕵️ Smart Analytics Tracking

Close deals by spying on how your leads interact with your proposal website concepts in real-time.

  • Opened Alerts: Logs instantly when they open your proposal link.
  • Heartbeats: Pings every 10 seconds to log reading duration.
  • Scroll Depth: Monitors how far down the website they scroll.
  • Checkout Clicks: Tracks clicks on checkout buttons.

Help: Getting Google API Keys

Option A: Google Custom Search API

  1. Enable the Custom Search API in your Google Cloud Console, create an API key, and paste it into the Google Custom Search API Key input.
  2. Go to the Google Programmable Search Console and create a search engine.
  3. In the search engine dashboard, turn ON "Search the entire web".
  4. Copy the Search Engine ID (CX ID) and paste it into the input.

Google Places API

  1. Go to the Google Cloud Console.
  2. Enable the Places API (New).
  3. Go to APIs & Services > Credentials, create an API Key, and paste it into the Places API Key input.

🚀 Deploying on Hugging Face

Because Hugging Face Space deployments are ephemeral, saved .env files do not persist across builds.

Configure your keys under Settings > Variables and Secrets in your Hugging Face Space dashboard as:

  • GOOGLE_PLACES_API_KEY
  • GOOGLE_SEARCH_API_KEY
  • GOOGLE_SEARCH_ENGINE_ID
  • GITHUB_USERNAME
  • GITHUB_REPO
  • GITHUB_BRANCH
  • GITHUB_TOKEN
  • DISCORD_WEBHOOK_URL
  • DISCORD_USER_ID (Optional)
